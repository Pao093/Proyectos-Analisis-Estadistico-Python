Proyectos de Análisis Estadístico con Python
Este repositorio contiene una colección de herramientas y scripts desarrollados bajo un enfoque de Ingeniería Estadística para la optimización de procesos, automatización de reportes y análisis de datos.

Proyecto destacable: Reporte de Inventario Automatizado
El script reporte_inventario_pandas.py es una solución diseñada para sustituir la gestión manual de datos por un flujo de trabajo automatizado utilizando la librería Pandas.

Funcionalidades técnicas:
Normalización y Limpieza: Preparación de estructuras de datos (DataFrames) para un análisis preciso.

Segmentación Avanzada: Clasificación de ítems mediante lógica condicional (Gamas de producto).

Cálculos Estadísticos: Generación de métricas clave como el valor total del inventario y promedios de precios.

Agrupamiento (Grouping): Resumen financiero por categorías para identificar áreas de mayor inversión.

Exportación de Resultados: Generación automática de reportes en formatos Excel y CSV.
