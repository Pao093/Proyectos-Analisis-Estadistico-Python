# Proyectos de Análisis Estadístico con Python

Este repositorio es un portafolio técnico de **Ingeniería Estadística** enfocado en la automatización de procesos, algoritmos de lógica y análisis de datos masivos.

## Proyectos Destacados

### 1. Reporte de Inventario Automatizado
* **Archivo:** reporte_inventario_pandas.py
* **Tecnología:** Python, Pandas.
* **Impacto:** Automatización de métricas de inversión y segmentación de productos por gama, sustituyendo procesos manuales por flujos eficientes.

### 2. Depuración y Optimización Logística
* **Archivo:** limpieza_datos_logistica.py
* **Tecnología:** Python (Data Wrangling).
* **Impacto:** Normalización de registros, eliminación de duplicados y ordenamiento jerárquico para auditorías físicas.

### 3. Simulador de Gestión Financiera
* **Archivo:** simulador_transacciones_logica.py
* **Tecnología:** Lógica de algoritmos y estructuras de control.
* **Impacto:** Implementación de protocolos de seguridad (validación de PIN) y generación de reportes estadísticos de sesión (ticket promedio).
